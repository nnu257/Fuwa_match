<概要>
・本プログラムは男性⇔女性の希望をポイント化し，お互いのポイント数が高いペアをマッチングします．

<詳細>
・片方だけが希望している場合にもマッチする
・ポイントが0の場合はマッチしない(=お互いが希望に入れていないときはマッチしない)
・同性マッチングはアリとする(その場合は希望してない人に許可が必要か．)

<使い方>
1. Pythonをインストールする
2. 希望順位を記述したファイルを用意する
    ・GoogleFormで用意する．
        ・自分の番号と希望順位n位までを聞く(nは自由だが，参加者全員でnを揃えること)
        ・csvでダウンロードする
        ・<参考>https://form.run/media/contents/googleform/google-form-download/
    ・念の為，ダウンロードしたcsv形式が以下のようになっていることを確認
        ・自分の番号に加えて，マッチングしたい人の番号をカンマ区切りで記述
        ・第一希望~第n希望まで記述されている
        ・1番の人の希望~20番の人の希望が上から順に記述されている必要はない
        ・<例> 3番の男性が14,13,16,17,17番の女性を希望する場合「3,14,13,16,17,18」
        ・<参考>preferecens_sample.csv
3. matching.pyのフラグを変更する
    ・お互いに相手を第n希望に書いているペアのみマッチングさせる場合→MUTUAL_PREFをTrueに，しないならFalseに
    ・同性のマッチングを許さない場合→AVOID_SAME_GENDER_MATCHINGをTrueに，しないならFalseに
4. マッチングを実行する
    ・python matching.py
     *同性のマッチングを許さない場合は，
      1.実行後に男性の人数を入力してください．
      2.男性からidをふるようにしてください．
